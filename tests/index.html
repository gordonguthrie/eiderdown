Showdown Compatibility Unit Test Generator For erlmarkdown

Purpose

erlmarkdown is a server-side implementation of markdown written in Erlang. In order to practically use it, you need a client side implementation (written in javascript) which produces the same outputs from the same inputs.

The client-side library of choice for erlmarkdown is showdown from Attack Labs.

It is not clear if showdown is still maintained - it is not available from a version control system. The current version we track is 0.9 and the source code is included with erlmarkdown.

This page just has a set of markdown terms in it which it runs through showdown to generate the unit tests that are included in erlmarkdown
